About AutoNest

AutoNest is a Java-based desktop car racing game built to demonstrate strong Object-Oriented Programming practices while delivering smooth, enjoyable gameplay. The project is implemented using Java (JDK 17+) and Java Swing/JavaFX for rendering.

OOP-focused
Collision System
GitHub Collaboration

Purpose

To learn and apply encapsulation, inheritance, polymorphism, and abstraction through a playable game that includes movement control and collision detection.

Core Features

  • Single-player race
  • Car controls: accelerate, steer
  • Collision & scoring
  • Start / Pause / Restart

Demo / How to run (developer notes)

  1. Clone the repo and open the Java project in your IDE (IntelliJ/Eclipse).
  2. Ensure JDK 17+ is installed.
  3. Run the main class (e.g., com.autonest.Main) or build with Maven/Gradle if configured.
If you need a packaged demo, we can add a runnable JAR in the /dist folder and document it in the README.
